TimeManage_ios
==========================
# 简介

  统计ios设备上的使用时间，与设备上各个应用的使用时间。
  本项目使用Objective-c进行开发，旨在实现对ios上应用的获取与后台线程的使用，目的是帮助用户直观得得到设备的使用时间和在各个应用上花费的时间。
